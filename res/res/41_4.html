Espressif provides the SMD module—ESP-WROOM-S2 that integrates ESP8266EX. The module has been adjusted to achieve the best RF performance. We recommend using ESP-WROOM-S2 for tests or for further development.

The flash used on this module is a 2-MB SPI flash connected to HSPI, with a package size of SOP 8-150 mil. The gain of the on-board PCB antenna is 2 dBi. The ESP-WROOM-S2 works as the SDIO/SPI slave with the SPI speed of up to 8 Mbps.

  • Home appliances
  • Home automation
  • Smart plugs and lights
  • Industrial wireless control
  • Image Recognition
  • Baby monitors
  • IP cameras
  • Sensor networks
  • Wearable electronics
  • Wi-Fi location-aware devices
  • Security ID tags
  • Retail & Catering Applications
  • Industrial Automation

Features:
  • Core: Xtensa® single-core 32-bit L106 CPU, frequency up to 160MHz
  • Memories
    • 160 KB of ROM
    • 1 KB of SRAM
    • 2 MB of Flash memory
  • Working Voltage: 3 V to 3.6 V
  • Up to 8 GPIOs
  • 1*10-bit ADC (up to 1 channel)
  • Communication interfaces
    • 1 I2C interfaces
    • 1 I2S interface
    • 2 SPI interfaces
    • 2 UART interfaces
  • Security:
    • 128 bit OTP
  • Extended temperature range: -40 to 85 °C
