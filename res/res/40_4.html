ESP-WROOM-5V2L is a Wi-Fi module integrates ESP8266EX, which supports IEEE802.11 b/g/n. ESP-WROOM-5V2L can communicate with other device by UART interface and can be applicated in smart home equipment, remote monitoring fields.

ESP-WROOM-5V2L module has a PCB antenna. This module has a 3.3 V LDO regulator ,can connect the logic voltage to support 5 V/3.3 V.

Applications:
  • Home appliances
  • Home automation
  • Smart plugs and lights
  • Industrial wireless control
  • Image Recognition
  • Baby monitors
  • IP cameras
  • Sensor networks
  • Wearable electronics
  • Wi-Fi location-aware devices
  • Security ID tags
  • Retail & Catering Applications
  • Industrial Automation
  • Smart POS Machines
Features:
  • Core: Xtensa® single-core 32-bit L106 CPU, frequency up to 160MHz
  • Memories
    • 160 KB of ROM
    • 1 KB of SRAM
    • 2 MB of Flash memory
  • Working Voltage: 3 V to 3.6 V
  • Up to 10 GPIOs
  • 1*10-bit ADC (up to 1 channel)
  • Communication interfaces
    • 1 I2C interfaces
    • 1 I2S interface
    • 2 SPI interfaces
    • 2 UART interfaces
  • Security:
    • 128 bit OTP
  • Extended temperature range: -40 to 85 °C
